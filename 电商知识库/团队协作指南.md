## 编辑文档（两种方式任选）

### 方式 A：网页直接编辑（新手推荐）

1. 打开仓库，进入 `电商知识库/痛点_问题_需求/职能痛点矩阵.md`
2. 点击右上角铅笔图标 ✏️
3. 直接修改内容
4. 页面底部填写修改说明（如：`补充广告投放痛点`）
5. 点击 `Commit changes`

### 方式 B：Claude Code 本地编辑（效率更高）

```bash
# 第一次：克隆仓库
git clone https://github.com/szqshan/eshop-ai-agent.git
cd eshop-ai-agent

# 每次编辑前先拉取最新内容
git pull

# 用 Claude Code 编辑（在项目目录里运行）
# ... 编辑文件 ...

# 编辑完推送
git add .
git commit -m "补充了xxx痛点"
git push
```

## 讨论问题

用 GitHub **Issues** 功能：
- 仓库页面 → Issues → New Issue
- 适合：提问、讨论方向、反馈问题
- 每个议题独立追踪，不会淹没在群消息里

---

## 注意

- 每次编辑前先 `git pull`，避免冲突
- 不确定怎么填，先看 `群成员必读.md` 里的示例
