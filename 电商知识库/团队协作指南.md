# 团队协作指南

> 文档托管在 GitHub，通过以下两种方式任选一种参与编辑。

---

## 方式 A：网页直接编辑（推荐新手）

不需要安装任何东西，在浏览器里就能改文档。

1. 打开 GitHub 仓库：`https://github.com/szqshan/eshop-ai-agent`
2. 进入 `电商知识库/痛点_问题_需求/职能痛点矩阵.md`
3. 点击右上角铅笔图标 ✏️
4. 直接编辑内容
5. 拉到页面底部，填写改了什么（如：`补充广告投放痛点`）
6. 点击 `Commit changes`

**几分钟后**公网文档自动更新。

---

## 方式 B：Claude Code 本地编辑（推荐技术型成员）

可以用 Claude Code 帮你写，效率更高。

### 第一次配置（只做一次）

```bash
# 1. 安装 Git（已有跳过）
# https://git-scm.com/download/win

# 2. 克隆仓库到本地
git clone https://github.com/szqshan/eshop-ai-agent.git
cd eshop-ai-agent

# 3. 配置你的身份
git config user.name "你的昵称"
git config user.email "你的邮箱"
```

### 每次编辑流程

```bash
# 第一步：拉取最新内容（每次编辑前必做）
git pull

# 第二步：用 Claude Code 打开项目编辑
# （在 eshop-ai-agent 目录里运行 claude）

# 第三步：编辑完成后提交并推送
git add .
git commit -m "补充了xxx痛点"
git push
```

---

## 注意事项

- **每次编辑前先 `git pull`**，避免冲突
- 提交信息简单说明做了什么即可
- 不确定怎么填内容，看 `群成员必读.md` 里的示例
- 遇到问题直接群里问
