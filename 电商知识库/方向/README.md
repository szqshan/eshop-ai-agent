## 评估维度

每个方向从以下维度打分（1-5分）：

| 维度 | 说明 |
|------|------|
| 痛点强度 | 卖家有多痛，愿意付多少钱解决 |
| 技术可行性 | 当前 AI 技术能否有效解决 |
| 开发成本 | 团队实现所需时间和资源 |
| 市场规模 | 潜在客户数量 |
| 竞争烈度 | 市场上已有多少类似产品 |
